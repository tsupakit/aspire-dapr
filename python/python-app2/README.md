# python-app2

A minimal FastAPI application served by Uvicorn.

## Run locally

1. Create a virtual environment:

```bash
python -m venv .venv
```

2. Activate it:

```bash
# Windows PowerShell
.\.venv\Scripts\Activate.ps1
```

3. Install dependencies:

```bash
pip install -e .
```

4. Start the app:

```bash
python main.py
```

5. Open:

- http://127.0.0.1:8000/
- http://127.0.0.1:8000/health
